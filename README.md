# EveOneSearch
## 这是什么?
这是一个奇妙搜索源的订阅，因为现在搜集订阅难，用户找订阅难，我们想要建设一个最完备的订阅，让用户**一个就够**!
## 使用方法
在奇妙搜索中导入订阅地址:<https://eos.oroc.top/EveOneSearch.txt>
